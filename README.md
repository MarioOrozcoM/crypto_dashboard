# 🚀 Crypto Dashboard

A modern cryptocurrency dashboard built with React and Tailwind CSS.

## 📸 Preview
![Preview](dashboard_cripto/.github/screenshot.png)

## ✨ Features

- Top 10 cryptocurrencies
- Live prices from CoinGecko API
- Search by name
- Auto refresh every 60 seconds
- Responsive design

## 🛠 Technologies

- React
- JavaScript
- Tailwind CSS
- CoinGecko API

## 🔌 API

Data provided by:
- CoinGecko API

## 🌐 Live Demo

https://crypto-dashboard-mario.netlify.app/

## ⚙ Installation

```bash
git clone https://github.com/MarioOrozcoM/crypto_dashboard.git

cd crypto_dashboard

npm install

npm run dev
```
